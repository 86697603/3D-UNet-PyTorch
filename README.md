# 3D-UNet-PyTorch
This repository is an implementation of [3D U-Net: Learning Dense Volumetric Segmentation from Sparse Annotation](https://arxiv.org/abs/1606.06650)

## Environment
+ Python 3.6
+ PyTorch 1.0
